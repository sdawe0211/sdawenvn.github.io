# 音樂系統測試說明

## 功能概述

### 1. 背景音樂 (chillhop.m4a)
- 在主選單、食譜選擇、遊戲畫面循環播放
- 音量：50%
- 可通過右下角按鈕控制開關

### 2. 成功音樂 (成功音樂.m4a)
- 在結算畫面播放
- 音量：60%
- 播放時會自動暫停背景音樂

## 測試步驟

1. **開啟遊戲**
   - 打開 virtual kitchen.html
   - 點擊頁面任何地方觸發音樂播放
   - 應該聽到背景音樂開始播放

2. **測試音樂控制按鈕**
   - 點擊右下角的音樂按鈕
   - 圖示應該從 🔊 變成 🔇
   - 顏色從青色變成紅色
   - 背景音樂應該停止

3. **測試結算畫面**
   - 進入遊戲並完成料理
   - 觸發 `window.showResultsScreen()` 函數
   - 背景音樂應該停止
   - 成功音樂應該開始播放

4. **測試重新開始**
   - 在結算畫面點擊「重新開始」按鈕
   - 成功音樂應該停止
   - 如果音樂開關是開啟的，背景音樂應該恢復播放

## 音樂控制 API

```javascript
// 檢查音樂是否啟用
window.musicControl.isMusicEnabled()

// 切換音樂開關
window.musicControl.toggleMusic()

// 播放背景音樂
window.musicControl.playBackgroundMusic()

// 暫停背景音樂
window.musicControl.pauseBackgroundMusic()
```

## 控制台日誌

正常運行時應該看到以下日誌：
- `🎵 Music control system initialized`
- `✅ Background music loaded and ready`
- `✅ Success music loaded and ready`
- `🎵 Background music started playing`
- `🎵 Background music paused for results screen`
- `🎉 Success music started playing`
